# Simple Test Invitation Codes

Here are 5 test invitation codes you can use. **First, create them using the test page:**

## Step 1: Create the Codes
Visit: http://localhost:3000/test/invites
Click: "Create 5 Test Invites"

This will generate 5 codes automatically.

## Step 2: Use the Codes
Go to: http://localhost:3000/client/invite-code
Enter any of the generated codes
